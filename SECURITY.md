# Security Policy

This repository is for OpenHuman Online documentation only.

## Do Not Report Secrets Publicly

Do not open public issues containing credentials, API keys, payment provider data, private customer data, screenshots with account details, or internal logs.

## Public Contact

Use the support contact shown on the live SaaS:

- https://openhuman.online/?utm_source=github&utm_medium=documentation&utm_campaign=openhuman_online_docs&utm_content=security_contact_home

## Scope

This docs project can accept corrections to public documentation. It is not a vulnerability disclosure inbox for private production infrastructure.
