# Checkout And Pricing

The public SaaS link for OpenHuman Online must always be published with UTM tracking.

## Default Plan

- Default plan: Team annual
- Pricing page: https://openhuman.online/pricing/?utm_source=github&utm_medium=documentation&utm_campaign=openhuman_online_docs&utm_content=checkout_pricing
- Checkout page: https://openhuman.online/checkout/?utm_source=github&utm_medium=documentation&utm_campaign=openhuman_online_docs&utm_content=checkout_checkout

## Link Rules

Use these fields unless a platform requires a different format:

- `utm_source=github`
- `utm_medium=documentation`
- `utm_campaign=openhuman_online_docs`
- `utm_content=<placement>`

## Good Placements

| Placement | Tracked URL |
| --- | --- |
| README hero | https://openhuman.online/?utm_source=github&utm_medium=documentation&utm_campaign=openhuman_online_docs&utm_content=readme_primary_home |
| Feature guide CTA | https://openhuman.online/?utm_source=github&utm_medium=documentation&utm_campaign=openhuman_online_docs&utm_content=feature_guide_cta |
| Pricing CTA | https://openhuman.online/pricing/?utm_source=github&utm_medium=documentation&utm_campaign=openhuman_online_docs&utm_content=pricing_cta |
| Checkout CTA | https://openhuman.online/checkout/?utm_source=github&utm_medium=documentation&utm_campaign=openhuman_online_docs&utm_content=checkout_cta |

## What Not To Publish

- Bare SaaS domains without UTM tracking.
- Private repository links.
- Cloudflare preview URLs unless the user explicitly wants a preview.
- Payment provider invoice URLs from live probes.
- Internal analytics, secrets, or customer-specific onboarding notes.
